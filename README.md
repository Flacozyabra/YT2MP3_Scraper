# YT2MP3 Scraper

Консольный Python-скрипт для автоматического скачивания аудиодорожек с YouTube-каналов, плейлистов или отдельных видео и их сохранения в формате MP3 (192 kbps).

## Ключевые особенности
- **Конвертация в MP3**: Автоматическое извлечение звука и перекодирование с помощью `ffmpeg` (оптимальное качество 192 kbps).
- **Защита от блокировок**: Случайные паузы (5–15 сек) между загрузками для защиты от банов со стороны серверов YouTube.
- **Обход DPI (для РФ)**: Полная маскировка TLS-отпечатков под браузер Chrome (`curl_cffi`) для корректной работы систем обхода блокировок (например, `zapret` или GoodbyeDPI).
- **Умный пропуск**: Ведение архива скачанных треков (`Music/archive.txt`) — повторные загрузки пропускаются мгновенно.
- **Поддержка куков**: Возможность импорта авторизации из популярных браузеров (Opera, Chrome, Firefox, Edge, Brave) для скачивания спонсорских или возрастных видео.
- **Отказоустойчивость**: До 10 повторных попыток скачивания при нестабильном сетевом соединении.

## Запуск
Просто дважды кликните по файлу `run.bat` в корне проекта.

## Особенности работы с ZAPRET (для пользователей в РФ)
Для стабильной загрузки видео без сбросов соединения (ошибки `curl: (35) Recv failure`) убедитесь в правильности настройки вашего `zapret`:
1. **Права Администратора**: Батник запуска `zapret` (обычно запускающий `winws.exe`) должен быть обязательно запущен **от имени Администратора**. Без этого драйвер `WinDivert` не сможет перехватывать и модифицировать трафик Python-скрипта.
2. **Правильный список хостов**: Домены YouTube должны обрабатываться «гугловской» стратегией обхода. Убедитесь, что следующие домены добавлены в файл **`list-google.txt`** (в папке `zapret\lists\`), а не в общий список `list-ultimate.txt`:
   ```text
   youtube.com
   www.youtube.com
   googlevideo.com
   *.googlevideo.com
   youtubei.googleapis.com
   ```
   *Если эти домены будут обрабатываться фейковым пакетом от других сайтов (например, `4pda.to` из `list-ultimate.txt`), сервера YouTube будут принудительно обрывать соединение.*
